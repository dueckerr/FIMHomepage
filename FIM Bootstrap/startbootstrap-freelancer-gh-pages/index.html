Data Collection, Storage, and Analysis

What differentiates PIM Tracker is the cloud based platform. Instantaneous collection, storage, and analysis of the maintenance logs, greatly reduces the work of combining multiple "Spreadsheets", while increasing the accuracy of the data, and spotting inaccurate entries.

A centralized database allows for machine learning capabilities. Machine learning provides systems the ability to automatically learn and improve from experience without being explicitly programmed. Machine learning focuses on the development of computer programs that can access data and use it learn for themselves.

Systematic Approach


With PIM Tracker, fracturing operations can begin using statistical data to make proactive decisions. Using simply statistical analysis methods, operators will have the means to determine the necessity of maintenance for each unit and part with factual data.

After enough maintenance records have been collected, we can show the probability of failure based on the standard deviation. By showing the probability of failure for each part and its corresponding stroke count, will allow the operators to make judgement calls on whether a single part should be replaced or the entire pump.
